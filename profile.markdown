## 🌟「映画」と「ゲーム」の、その先へ。

高校時代に映画同好会を立ち上げ、大学を卒業するまで映画づくりに没頭していました。

その後、ゲーム会社に入社したのをきっかけに個人でもゲーム制作を始め、今は恋愛ゲームをつくっています。

映画からゲーム、その先はまだ誰も見たことがない、「**新しい物語**」をつくっていきます。

### ターミネーター2
間違いなく人生で一番観た映画。すべてはここから始まりました。

{% include comment.html character="haruka" comment="ターミネーター2が好きすぎて、自分で液体金属ターミネーター役をやって一人で短編映画を撮ったらしい。" %}
